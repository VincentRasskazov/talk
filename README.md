# Talk

A Discord‑style real‑time messaging app built with React and Firebase. Includes servers, channels, direct messages, and WebRTC voice/video calling.

## Features
- Servers with text channels and invite codes
- Direct messaging with read receipts
- URL previews, inline images, and GIF support
- Peer‑to‑peer WebRTC voice/video calls
- Role‑based permissions and moderation tools
- Offline caching and optimized Firebase reads
- Optional AI assistants via trigger tags

## Tech Stack
- React.js, CSS
- Firebase Firestore and Firebase Auth
- Microlink (URL previews), Tenor API (GIFs)
- GitHub Pages + GitHub Actions for deployment

## Local Development
```bash
git clone https://github.com/VincentRasskazov/talk.git
cd talk
npm install
npm start
```

Add your own Firebase configuration to enable backend features.
